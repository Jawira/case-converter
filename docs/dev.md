Development
===========

Testing
-------

- Test all:

    ```bash
    $ phing qa
    ```

- PHPUnit:

    ```bash
    $ phing phpunit:run
    ```

- Behat:

    ```bash
    $ phing behat:run
    ```

- Php lint:

    ```bash
    $ phing php:lint
    ```

- Validate Composer:

    ```bash
    $ phing composer:validate
    ```
