Development
===========

PHPUnit
-------

Install PHPUnit:

```bash
$ make bin/phpunit
```

Running PHPUnit:

```bash
$ bin/phpunit
```

Behat
-----

Install Behat:

```bash
$ composer install
```

Run behat tests:

```bash
$ vendor/bin/behat
```
